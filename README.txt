Proyecto estático de ejemplo para Farmaceus con Bootstrap 5

- Todas las imágenes del ZIP original fueron optimizadas (redimensionadas y comprimidas)
  y se encuentran en: assets/img/
- El fondo general del sitio usa la imagen de píldoras (pexels-shvetsa-3683041.jpg) con
  una capa traslúcida para no afectar la lectura.

Estructura:
- index.html
- productos.html
- catalogo.html
- servicios.html
- contacto.html
- assets/css/custom.css
- assets/js/main.js
- assets/img/*.jpg, *.png (imágenes optimizadas)

Uso:
1. Abre la carpeta 'farmaceus-bootstrap-site' en Visual Studio o Visual Studio Code.
2. Abre 'index.html' en el navegador o usa una extensión tipo Live Server.
3. Puedes subir todo a GitHub Pages para tener un sitio estático público.
